# 材料
  - やきそば麺 120g
  - ベースソース 20cc
  - ごま油 少量
  - 花椒 大量
  - ニラ 40g
  - もやし 40g
  - 長ネギ 40g
  - 豚小間 30g

# 激辛パウダー
  - カエンペッパー 5g
  - ハバネロ 3g

# レシピ
  1. サラダ油(分量外)をひいたフライパンでスパイス類を弱火で炒めます。
  2. 別のフライパン豚小間を炒めます。
  3. もやし・キャベツ・麺をのせて、約2分半蒸し焼きにします。
  4. ベースソースと1で作ったスパイスを絡めます。
  5. 3にソースを混ぜ、お好みで大量に花椒をかけます。
